# Tick-Tack-Toe-Android-Studio

Tic-tac-toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. To add this project to android studio.

# Features
1) Single Player with Difficulty levels
2) Two Player
# Getting started
1) In android studio goto file
2) Select new > Import from version control
3) Choose git / github (git option recommanded )
4) Paste url to repository i.e https://github.com/GRaghuNandanLal/Tick-Tack-Toe-Android-Studio
5) Click on clone.
